# 英検アプリ GitHub Pages 公開手順（非エンジニア向け・Webだけで完結）

作成日：2026-06-02 ／ コマンド操作なし・ブラウザだけでできます。

---

## STEP 0：GitHubアカウントを持っているか確認

以下のいずれかで確認できます。

1. ブラウザで https://github.com/login を開く
2. メールアドレス（`j.sakamoto@gw.sakuramegane.com` や個人用メール）でログインを試す
   - ログインできる → アカウントあり。STEP 2へ
   - パスワードを忘れた → 「Forgot password?」でリセット
3. それでも不明なら、メールアプリで「GitHub」を検索 → 過去に登録メールが届いていればアカウントあり

### アカウントが無い場合（STEP 1）
1. https://github.com/signup を開く
2. メールアドレス・パスワード・ユーザー名（例：`sakamoto-arcus`）を入力
   ※ユーザー名は公開URLの一部になります（半角英数字・ハイフン）
3. 確認コードがメールに届く → 入力
4. 無料プラン（Free）を選択して完了

---

## STEP 2：リポジトリ（置き場所）を作る

1. ログイン後、右上の「＋」→「New repository」
2. Repository name：`eiken5`（半角英数字。何でもOK）
3. 公開設定：**Public**（GitHub Pagesの無料公開にはPublicが必要）
4. 「Add a README file」は **チェックしない**
5. 「Create repository」をクリック

---

## STEP 3：アプリのファイルをアップロード

1. 作成されたリポジトリ画面で「uploading an existing file」リンクをクリック
   （または「Add file」→「Upload files」）
2. パソコンで `G:\マイドライブ\claude\英検アプリ\` フォルダを開く
3. 以下の**5つのファイルを選んでドラッグ＆ドロップ**（これだけで動きます）：
   - `index.html`
   - `words.js`
   - `manifest.json`
   - `sw.js`
   - `icon.svg`
   ※ `data` フォルダや README は無くても動作します（入れてもOK）
4. 下の「Commit changes」ボタンを押す

---

## STEP 4：GitHub Pages を有効にする

1. リポジトリ上部の「Settings」タブをクリック
2. 左メニューの「Pages」をクリック
3. 「Branch」で `main` を選び、フォルダは `/ (root)` のまま「Save」
4. 1〜2分待つとページ上部に公開URLが表示される：
   ```
   https://（ユーザー名）.github.io/eiken5/
   ```

---

## STEP 5：iPadのホーム画面にアプリとして追加

1. iPadの **Safari** で STEP 4 のURLを開く
2. 画面下（または上）の「共有」ボタン（□に↑）をタップ
3. 「ホーム画面に追加」をタップ → 名前を確認して「追加」
4. ホーム画面にアイコンができ、タップするとアプリのように全画面で起動

---

## 困ったとき
- ページが表示されない：公開直後は数分かかる。URLのスペル（末尾の `/` 含む）を確認
- 音声が出ない：最初の問題で「🔊 もういちど きく」を一度タップ
- 更新したのに古いまま：Safariの場合、一度アプリを閉じて開き直す（サービスワーカーのキャッシュ更新）

## 単語を増やした・直したとき（再公開）
1. パソコンで `data/words.json` を編集
2. `words.js` を再生成（README.md 参照）
3. GitHubのリポジトリで該当ファイルを「Upload files」で上書きアップロード
